# Architecture Decision Records (ADRs)

This directory holds Architecture Decision Records — short documents capturing a
significant architectural or design decision, its context, and its consequences.

## Format

Create one Markdown file per decision, numbered sequentially:

```
docs/adr/0001-short-title.md
```

A simple template:

```markdown
# NNNN. Title

- Status: proposed | accepted | superseded by ADR-XXXX
- Date: YYYY-MM-DD

## Context

What is the issue we're addressing and what forces are at play?

## Decision

What did we decide to do?

## Consequences

What becomes easier or harder as a result?
```

See [Michael Nygard's ADR pattern](https://cognitect.com/blog/2011/11/15/documenting-architecture-decisions)
for background.
